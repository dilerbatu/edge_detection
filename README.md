# edge_detection
Edge Detection using C++ and Python

Installation

```
$ git clone https://github.com/dilerbatu/edge_detection.git
```

```
$ cd /yourpath/edge_detection
```

```
$ cmake .
$ make
```


For Running C++ code
```
$ ./edge_detection_bin
```



For Running python code

```
python3 edge_detector.py
```
